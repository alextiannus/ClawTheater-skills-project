# 场景工艺

> 小说是由场景构成的。一个场景不是"发生了什么事"，而是"一个角色想要某样东西，遇到了阻碍，结果比预期更糟"。本技能覆盖场景内部的每一个工程细节。

---

## 一、场景结构：目标-冲突-灾变

来自 Dwight Swain 的经典框架。每个场景（Scene）和过渡（Sequel）交替出现。

### Scene（行动场景）

```
目标（Goal）：角色在这个场景中想要什么？
  ↓
冲突（Conflict）：什么阻碍了他？
  ↓
灾变（Disaster）：结果比他预想的更糟。
```

灾变不一定是灾难。它可以是"得到了想要的，但代价超出预期"，或者"问题解决了，但引出了更大的问题"。关键是：场景结束时角色的处境不能比开始时更好（除非是全书高潮）。

**灾变的四种形态：**
- "不。"——直接失败。
- "是的，但是……"——成功了，但附带了严重后果。
- "不，而且……"——不仅失败了，情况还变得更糟。
- "是的！"——只在极少数关键转折点使用。

### Sequel（过渡场景）

```
反应（Reaction）：角色对灾变的情绪/身体反应。
  ↓
困境（Dilemma）：摆在面前的选择都不好。
  ↓
决定（Decision）：角色做出选择——这个选择成为下一个Scene的目标。
```

Sequel可以很短（一个段落），也可以很长（一整章的沉思和对话）。节奏快的部分，Sequel压缩到一两句。节奏慢的部分，Sequel展开来写，用对话和日常场景填充。

### 实战示例

```
Scene:
目标：林启想在限时内送完这单外卖。
冲突：骑到桥中间，掌心突然跳了一下，他被迫停车查看。
灾变：什么都没看到——但时间被耽误了，而且这种感觉越来越不对。

Sequel:
反应：他攥了一下拳，松开。跳停了。
困境：继续跑单（忽略异常）还是停下来研究掌心？
决定：继续跑。但他开始在地图上标记每次掌心跳动的位置。
     （这个决定成为下一个Scene的目标。）
```

## 二、对话工艺

### 潜台词：说的不是想的

好的对话中，角色很少直说自己想要什么。他们暗示、试探、回避、撒谎、言不由衷。

**四种潜台词技法：**

**1. 答非所问。** 角色回避了问题，回答了另一个。

```
"你害怕吗？"
"你今天吃了吗？"
——他不想谈害怕。但他的回避本身就是答案。
```

**2. 用行动替代回答。** 角色不回答，而是做了一件事。

```
"中间饿了吃。"
"我不——"
"带上。"
——林建国没有说"我担心你"。他做了一份吐司。
```

**3. 说反话。** 角色说的和感受的相反。

```
"没事。"
"你脸都白了。"
"我说没事。"
——读者知道他有事。角色也知道对方知道。但他就是不说。
```

**4. 沉默和中断。** 角色没说完的话、突然中断的对话、长时间的沉默。

```
"昨天下午很明显的——"
"可能是什么工程。"
——林建国打断了话题。他不想讨论天上那个灰点。
   但他到底是不关心，还是不想让儿子关心？
   读者不确定。这就是张力。
```

### 对话节奏

- 短句传递紧张感。"走了。""看路。""知道。"——每个字都有重量。
- 长句传递犹豫或思考。"从淡滨尼看得很清楚。"——他在试图说服。
- 两个角色的句子长度应该不同——这是区分角色声音的最快方式。
- 动作穿插在对话之间，一句话搞定。"林建国拧了一下火。"不要写"林建国慢慢地伸出手，拧了一下灶台上的火焰调节旋钮。"

### 去除对话标签

"说道""回答道""问道"——能省就省。用动作替代。

```
标签：他沉吟了一下说道："我不确定。"
替代：他沉吟了一下。"不确定。"
```

```
标签："你确定吗？"他紧张地问道。
替代："你确定？"（紧张不用说，前文的场景已经营造了。）
```

## 三、张力递进

### 递进原则

同类事件每次出现时，强度必须递增。

```
第一次：掌心温了一下。很轻。
第二次：掌心烫了一下。一秒不到。像摸了铁栏杆。
第三次：掌心跳了一下。咚。不是温——是跳。
第四次：掌心猛跳——整条右臂跟着震——
第五次：连续的——咚咚咚咚——像手机开了震动——整条右臂从指尖到肩膀都在发麻
```

每一次都比上一次更强、更具体、更让角色（和读者）不安。如果中间有一次"退步"（比如从跳退回温），需要有叙事理由（比如角色远离了水源）。

### 间隔原则

两个高张力场景之间必须有一个低张力的"呼吸"段落。

```
高：掌心在天桥上猛跳，金色的点出现又消失。
低：法里德路过，聊了几句，骑走了。
高：芽笼河边，连续跳动，右臂发麻到停车。
```

没有"低"的间隔，"高"会失去冲击力。读者需要从紧张中喘口气，下一波紧张才会更有效。

### 期待感 vs 悬念

**悬念**：读者和角色都不知道会发生什么。（掌心为什么会跳？）

**期待感**：读者知道一些角色不知道的事。（读者已经发现六个红点都在水边，但角色还没发现。）

两者交替使用。纯悬念会让读者焦虑，纯期待感会让读者着急。最好的节奏是：悬念→角色开始调查→读者比角色先看出规律（期待感）→角色终于发现→新的悬念。

## 四、节奏控制

### 句子长度 = 速度

- 短句 = 快。"翻开右手。掌心正中间。有东西。"
- 长句 = 慢。"他坐在海堤上，掏出手机打开地图，六个红点分散在新加坡岛的中部和东部。"

紧张时用短句。日常时用长句。两者交替产生节奏感。

### 段落密度 = 呼吸

- 一句话一段 = 极度强调。"温的。"
- 多句话一段 = 正常叙事。

重要的转折或发现，单独成段。日常描写可以合并。

### 场景切换 = 跳跃

场景之间用分割线（---）断开。切换越频繁，节奏越快。一个场景写得越长越深，节奏越慢。

快节奏章节：3-4个场景，每个场景500-800字。
慢节奏章节：1-2个场景，每个场景1500-2500字。

## 五、章末钩子

每章结尾必须让读者想翻下一页。三种钩子类型：

### 悬念钩子

揭示一个新的谜团，不给答案。

```
掌心里的金色的点跳了一下。
像在回答。
```

读者的问题：这个点是什么？它有意识吗？——翻页。

### 反转钩子

最后一句推翻前面建立的认知。

```
他以为灰点是工程的反光。
然后它动了。
```

### 信息差钩子

给读者一个角色还不知道的信息。

```
林启骑车离开了。
他身后，芽笼河的水面上，有什么东西在往上浮。
```

读者知道了，但角色不知道。读者急着想看角色什么时候会发现。

### 钩子的节制

- 不要每章都用同一种钩子。三种轮换。
- 钩子必须在后续章节中兑现。挖了坑不填，读者会失去信任。
- 钩子不需要很大。"温的。"两个字就是一个钩子——因为读者想知道"温的"意味着什么。

## 六、时间过渡

### 允许的过渡方式

```
"第二天。"
"下午六单。"
"中午没回kopitiam。"
```

简洁、功能性、不解释。

### 禁止的过渡方式

```
"时间过得很快，转眼已经是第二天了。"（作者感叹）
"经过了一个漫长的下午之后……"（作者评价）
"就这样，一天过去了。"（作者总结）
```

## 七、感官沉浸

每个场景至少激活两种感官。不要只写"看到"。

```
视觉：晚霞把海面烧成橙色。
听觉：远处巴刹里的推车声、水声、有人喊"鱼来了"。
触觉：空气黏在皮肤上。
嗅觉：参巴辣酱的味道从缝里钻出来。
味觉：咖椰酱的甜味和黄油的奶香在嘴里化开。
```

感官细节必须通过角色体验。不要写"空气中弥漫着一种令人愉悦的香气"——这是导游在介绍。写"辣得他肚子叫了一声"——这是角色在闻。

## 自检清单

每个场景写完后：

- [ ] 场景有目标吗？角色想要什么？
- [ ] 有冲突吗？什么阻碍了他？
- [ ] 结束时角色的处境变了吗（通常是更糟）？
- [ ] 对话有潜台词吗？角色是否说了他不想说的？
- [ ] 张力是递进的吗？比上一次更强？
- [ ] 高张力之间有呼吸的空间吗？
- [ ] 章末有钩子吗？读者想翻下一页吗？
- [ ] 至少激活了两种感官吗？
- [ ] 所有感官细节都是角色正在体验的吗？
