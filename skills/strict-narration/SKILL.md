# 严格叙事规范

> 本技能解决一个问题：**让每一句话都像角色正在经历的事，而不是作者在旁边讲解。**
>
> 这是AI写作中最顽固的缺陷。AI天然倾向于"解释"和"总结"——因为训练数据中充满了百科、教程、评论。本技能的核心是一套严格的检测与修正流程，把"讲解模式"切换为"经历模式"。

---

## 核心原则：摄像机，不是导游

想象叙事者是一台绑在角色肩膀上的摄像机。它能拍到角色看到的、听到的、摸到的、闻到的。它能捕捉角色的动作和对话。但它不能：

- 飞到空中拍全景然后解释"这座城市有三百万人口"
- 暂停画面然后弹出字幕卡介绍"此人名叫张三，今年35岁，是一名程序员"
- 在角色做完一件事后插入旁白"由此可见他很聪明"
- 跳到另一个角色的视角解释"他不知道的是……"

这就是"摄像机"和"导游"的区别。摄像机只记录当下。导游会停下来给你讲背景知识。

**小说叙事必须是摄像机。**

## Deep POV（深层第三人称视角）

Deep POV = 第三人称写法，但具有第一人称的沉浸感。叙事紧贴一个角色，读者通过这个角色的感官和认知来经历故事。

### 操作规则

**1. 消除过滤动词。** "他看到""他听到""他感觉到""他注意到"——这些词在读者和体验之间插了一层玻璃。

```
过滤：他看到门开了，一个女人走了进来。
直接：门开了。一个女人走了进来。
```

```
过滤：他感觉到掌心在发烫。
直接：掌心烫了一下。
```

**2. 消除思维标签。** "他想""他觉得""他意识到""他知道"——这些是作者在报告角色的心理活动，而不是让读者直接进入。

```
标签：他觉得时间还算宽裕。
直接：时间还算宽裕。（叙述本身就代表角色的判断）
```

```
标签：他知道父亲不会改变主意。
直接：父亲不会改变主意。
```

**3. 叙述=角色的声音。** 在Deep POV中，叙述的语气、用词、节奏应该像角色本人在说话。一个粗犷的外卖骑手不会用"温馨""雅致"这种词。他的世界是"热""辣""累""酸"。

## 画外音：六种检测模式

以下是画外音的六种具体形态。写完每一段后逐条检查。

### 模式一：作者评论

作者对角色或事件发表评价、判断、感叹。

```
画外音：他父亲做白斩鸡这件事上，二十年没失过手。
```
"二十年没失过手"是作者在替读者做判断。角色在这一刻想的不是"我爸做鸡做了二十年"——他在吃鸡。

```
修正：鸡皮完整的，没有一点破裂。
```
直接写鸡的状态。读者自己判断手艺好不好。

```
画外音：这是他一天里最奢侈的一块五毛钱。
修正：（删掉。前后场景已经展示了他的经济状况。）
```

```
画外音：不是问句。
修正：（删掉。"坐。吃了再去。"本身的语气已经传达了一切。）
```

### 模式二：背景介绍

作者暂停叙事，插入一段背景信息。

```
画外音：法里德。马来人，也跑唐人街外卖的，同一片区经常碰到。
         他骑的是电动自行车——比林启的铁驴贵了八倍——
         但每个月赚的也没比林启多多少。
```
三句话全是作者在给读者念人物简历。

```
修正：法里德。电动自行车，唐人街外卖的保温箱，同一片区经常碰到。
```
只留角色此刻能看到的东西（车、保温箱），加一个关系标记（经常碰到）。其余通过后续对话自然展开。

### 模式三：概述总结

作者用几句话概括一段时间或一系列事件，而不是挑一个具体场景来写。

```
画外音：这些小费加起来，和他在麦当劳凌晨班的时薪加起来，
         再和他偶尔帮牛车水五金店搬货的日结加起来，
         刚好够交学费、付房租、吃饭——
         前提是吃饭每天控制在八块以内。
         他每个月还要给家里一千二。
```
这是一份财务报告，不是叙事。

```
修正：（删掉。通过具体场景展示经济状况——他吃四块五的杂菜饭、
         喝一块五的100Plus当犒劳、骑一辆破自行车。读者自己拼出全貌。）
```

### 模式四：常识科普

作者插入关于世界/城市/文化的一般性知识。

```
画外音：新加坡的双层巴士上层永远比下层凉，因为离空调出风口近。
```
这是百度百科，不是角色的体验。

```
修正：他坐在巴士上层靠窗的位置。（凉不凉让角色的行为说话——他选了上层。）
```

```
画外音：新加坡的热不分早晚——从水泥里蒸上来，从沥青里蒸上来，
         从组屋的外墙里蒸上来。
```
"新加坡的热不分早晚"是作者在做气象总结。

```
修正：空气黏在皮肤上，T恤后背已经贴住了。
```
角色此刻感受到的。

### 模式五：情绪告知

直接告诉读者角色的情绪状态。

```
画外音：他很害怕。
修正：他的手在抖。（让读者自己判断是害怕。）
```

```
画外音：她心中充满了感激。
修正：她低下头，过了好一会儿才抬起来。"谢谢。"声音很轻。
```

### 模式六：选择性叙述标注

作者用标注式语言点出一个观察结论。

```
画外音：选择性的。只在某些地方跳，某些地方不跳。
修正：（删掉。前面已经写了"后港没跳""淡滨尼没跳""宏茂桥跳了"——
         读者自己会发现"选择性"这个规律。）
```

```
画外音：这一单最邪门。
修正：（删掉。直接写体验的剧烈程度，读者自己判断"邪门"。）
```

## MRU：动机-反应单元

Dwight Swain的核心原则：**永远先写原因，再写反应。反应的顺序遵循人类的真实处理序列。**

### 结构

```
动机（外部事件）
  ↓
反应：
  1. 感受/本能反应（最快——身体先于意识）
  2. 非自主身体反应（攥拳、后退、屏息）
  3. 有意识的动作（打开手、站起来、看过去）
  4. 语言（最慢——说话需要组织语言）
```

### 示例

```
错误顺序：
"什么东西。"他自言自语。掌心猛跳了一下。

正确顺序：
掌心猛跳了一下。（动机）
整条右臂跟着震了一下。（本能反应）
他攥紧了车把。（非自主身体反应）
"什么东西。"（语言——最后）
```

### 省略原则

不是每次都要写全四步。快节奏时可以跳过中间步骤：

```
掌心跳了一下。（动机）
他攥了一下拳，松开。（直接动作）
```

但顺序不能反。永远不能先写"他攥了拳"再写"因为掌心跳了"。因果倒置会打破沉浸感。

## 信息传递：场景化原则

背景信息、世界设定、角色历史——这些都必须通过场景进入故事。

### 技法一：用对话传递

```
需要传达：林启有三份工，经济很紧张。

画外音方式：这些小费加起来……刚好够交学费。
场景方式：
"你今天几单？"法里德问。
"六单。晚上还有麦当劳的班。"
"你什么时候睡觉？"
"周末。"
```

三句对话，读者知道了：多份工、没时间睡觉、但语气轻松（性格）。

### 技法二：用行为传递

```
需要传达：这个kopitiam开了二十六年，菜单从没变过。

画外音方式：林建国做这套早餐做了二十六年。菜单很短——kopi、kopi-o……
场景方式：
陈叔坐下。"Kopi-o。"林建国说"知道。"
——这两个字比任何解释都有力。一个"知道"就说明了一切。
```

### 技法三：用物件传递

```
需要传达：林建国的性格——方正、严谨。

画外音方式：林建国的字，方方正正的……这个男人写了一辈子的字。
场景方式：
保鲜盒旁边有一张便条。
饭在桌上。我去进货。晚。——爸
方方正正的字，每个字之间隔得很开。
——便条本身就是人物。不需要作者点评。
```

## 自检流程

每写完一段话，问三个问题：

**问题一：这句话去掉角色还能成立吗？**
- "新加坡的热不分早晚" → 去掉角色，照样成立 → **画外音，删掉**
- "空气黏在皮肤上" → 去掉角色，没有皮肤可黏 → **角色体验，保留**

**问题二：这是角色此刻正在经历的吗？**
- "他花两百块从退休骑手那里买的这辆车" → 此刻他在骑车，不是在回忆买车 → **画外音**
- "右刹车捏到底才有反应" → 此刻他正在捏刹车 → **当下体验，保留**

**问题三：我在解释还是在呈现？**
- "他父亲做白斩鸡二十年没失过手" → 在解释 → **画外音**
- "鸡皮完整的，没有一点破裂" → 在呈现 → **保留**

## 常见陷阱与修正

| 陷阱 | 示例 | 修正 |
|------|------|------|
| "他觉得/他想" | 他觉得时间够用 | 时间够用 |
| "这是因为" | 他停下来，这是因为手在发麻 | 手在发麻。他停下来 |
| "他不知道的是" | 他不知道的是，灰点比昨天大了 | （删掉。从另一个角色的场景中展示） |
| "N年前/N年来" | 二十六年的肌肉记忆 | （通过行为的精准和流畅来展示） |
| "别人摸不出来" | 别人摸不出来，只有他自己知道 | （删掉。只写"温的"。读者知道这是他的私密体验） |
| "其实/事实上" | 其实他心里很担心 | 他把筷子放下了。过了一会儿又拿起来 |
| 修辞性评论 | 这两个字没有第二种理解方式 | （删掉。对话本身已经足够有力） |

## 唯一的例外

**时间跳跃和场景过渡**可以用简洁的叙述完成，不需要场景化：

```
允许：第二天。
允许：下午六单。
允许：中午没回kopitiam。
```

这些是结构性过渡，不是画外音。它们的功能是"快进"，而不是"解释"。判断标准：如果去掉这句话读者会困惑（不知道时间/地点变了），就保留。如果去掉不影响理解，就删。
