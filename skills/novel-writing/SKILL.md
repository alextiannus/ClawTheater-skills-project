# 小说创作四层框架

> **技能地图：** 本技能是总纲。写作时按需加载以下专项技能：
>
> | 技能 | 用途 | 何时加载 |
> |------|------|---------|
> | `strict-narration` | 叙事声音：Deep POV、画外音检测、MRU因果 | **每次写正文时必须加载** |
> | `scene-craft` | 场景工艺：结构、对话、张力、钩子 | 写具体场景时 |
> | `bestseller-structure` | 故事架构：四幕、节拍、弧线、系列设计 | 规划全书/全卷结构时 |
> | `reader-psychology` | 读者心理：上瘾机制、爽点、信息差、代入感 | 检查吸引力/设计爽点时 |
> | `novel-writing-advanced` | 进阶技法：人物深度、因果链、潜台词 | 打磨人物和情节时 |
> | `zhouzi-style` | 肘子风格：气人主角、系统漫才、轻松幽默 | 需要肘子风格时 |

好小说是四层结构，从底层到顶层依次构建。每一层都依赖下面的层，上层不能与下层矛盾。创作时自底向上设计，写作时从顶层呈现——**读者只看到最上面一层**。

```
┌─────────────────────────────────────────┐
│  第四层：讲故事的方法                      │  ← 读者看到的
│  叙事视角、节奏、悬念、场景呈现              │
├─────────────────────────────────────────┤
│  第三层：剧情设定                          │  ← 作者编排的
│  故事线、冲突、高潮、伏笔                   │
├─────────────────────────────────────────┤
│  第二层：人物设定                          │  ← 故事的灵魂
│  性格、立场、利益、行为逻辑                  │
├─────────────────────────────────────────┤
│  第一层：世界观 / 真实世界模型               │  ← 一切的地基
│  物理规律、社会结构、科技水平、文化           │
└─────────────────────────────────────────┘
```

**核心铁律：作者从上帝视角知道所有四层，但绝不能在故事中直白告诉读者。故事的魅力在于让读者通过第四层的阅读体验，自己去发现第三层的剧情走向、第二层的人物本质、第一层的世界真相。**

## 创作流程

每次开始一个新的小说项目，按以下顺序推进：

```
第一步：需求确认（类型、风格、篇幅、目标读者）
    ↓
第二步：构建世界观（第一层）
    ↓
第三步：设计人物（第二层）——人物必须在世界观中合理存在
    ↓
第四步：编排剧情（第三层）——剧情由人物的性格和利益冲突自然推动
    ↓
第五步：章节创作（第四层）——用讲故事的方法把一切呈现给读者
    ↓
第六步：质量检查——逐层验证一致性
```

### 需求确认

使用 AskUserQuestion 工具确认以下信息：

```
问题 1：小说类型
- 玄幻奇幻（武侠、仙侠、玄幻）
- 都市言情（职场、校园、豪门）
- 悬疑推理（侦探、惊悚、犯罪）
- 科幻末世（未来、机甲、末日）
- 历史军事（穿越、权谋、战争）
- 其他（让用户描述）

问题 2：篇幅长度
- 短篇（1-3 万字，完整故事）
- 中篇（5-10 万字，有起承转合）
- 长篇连载（分章节，10 万字以上）

问题 3：故事风格（可多选）
- 轻松幽默 / 热血爽文 / 感人催泪
- 烧脑反转 / 黑暗现实 / 温暖治愈

问题 4：主角设定
- 性别、年龄、身份
- 性格特点
- 特殊能力/金手指
```

## 第一层：世界观 / 真实世界模型

世界观是一切的地基。它定义了故事发生的"物理空间"——不仅是地理环境，更包括社会运作规则、科技水平、文化背景、权力结构。

### 核心原则

**准确性是底线。** 如果故事设定在现实世界或现实世界的延伸，那么物理学、数学、经济学、社会学等基础学科的常识不能违背。读者会原谅魔法和超能力，但不会原谅重力失效或货币体系不合逻辑。

**内部一致性。** 即使是完全虚构的世界，其规则必须自洽。修炼体系有上限就不能随意突破，经济系统有成本就不能凭空生产。设定了规则就要遵守，否则故事的张力会崩塌。

**世界观为人物和剧情服务。** 不要为了"酷"而堆砌设定。每一条设定最终要么约束人物行为、要么推动剧情发展、要么制造冲突。用不到的设定就不要写。

### 构建清单

无论什么类型的小说，世界观都应该明确以下几方面：

**物理/自然层面**——这个世界的"自然法则"是什么？有修炼体系？有超能力？还是纯现实物理？如果有特殊设定，其代价和限制是什么？

**社会/政治层面**——权力结构是什么？谁统治谁？社会阶层如何划分？底层人如何生存？顶层人如何维持权力？

**经济/科技层面**——科技水平到哪一步？经济怎么运转？普通人的日常生活是什么样的？钱从哪来？

**文化/语言层面**——人们说什么语言？信仰什么？有什么社会习俗？文化冲突在哪里？

**时间线**——故事发生在什么时间？之前发生过什么重大事件？世界正在朝什么方向变化？

### 不同类型的侧重

**现实/都市题材**：世界观就是当下的现实世界，必须符合真实的科技水平、公司运作、法律常识、社会结构。涉及真实公司和技术时要查资料。涉及城市时要写对地名、交通、饮食、气候、语言习惯。

**科幻题材**：科幻的"幻"建立在"科"的基础上。成本数字要有根据（硬件成本、能源消耗、AI推理费用等应基于真实数据的合理外推）。技术发展路径要符合趋势。

**玄幻/仙侠题材**：修炼体系是世界观的核心。境界划分要清晰、差距要明确、晋级代价要合理。宗门/世家/帝国的势力版图要画清楚。

**历史题材**：大历史背景必须准确，重大事件走向不能改。小细节可以虚构。历史人物性格要符合史实。要平衡历史真实性和故事趣味性。

### 世界观展示原则（重要——与第四层衔接）

**融入日常，绝不说教。** 世界观通过角色的生活自然展示，不是用旁白介绍。

坏的写法："到了2033年，全球92%的车辆使用AI驾驶系统，机器人保有量超过四千万台……"

好的写法：主角叫了一辆车，车自己开过来，他进去说了个地址，车就走了。他在后座用手机养龙虾。路上经过一个工地，全是机器人在干活，没有一个人类工人。他习以为常。——世界观就在这些细节里。

## 第二层：人物设定

人物是故事的灵魂。**所有剧情都应该由人物的性格、背景、利益和价值观自然驱动，而不是作者强行安排。**

### 核心原则

**每个人物都是一个完整的人。** 他有自己的性别、年龄、文化水平、社会地位、世界观、人生观和价值观。他的利益和兴趣所在决定了他的立场和行为准则。在任何一个剧情节点上，他的行为都应该合乎他的人设——如果你站在他的立场上，考虑他所知道的信息，你也会做出类似的选择。

**没有工具人。** 每个重要配角都有自己的目标和动机，不是为了服务主角而存在。反派也有自己的逻辑和苦衷——没有一个角色应该"为了作恶而作恶"。

**立场可以变化，但变化必须合理。** 人会因为利益变化、认知更新、情感冲击而改变立场。但变化需要铺垫，要符合人物性格，不能突兀。

### 人物设定模板

每个重要角色在写正文之前，必须完成以下设定：

```markdown
## 人物档案：[姓名]

### 基本信息
- 姓名 / 年龄 / 性别
- 外貌特征（1-2个最突出的辨识点）
- 身份地位（职业、社会阶层、在故事世界中的位置）

### 背景与经历
- 家族/出身
- 关键成长经历（塑造性格的事件）
- 当前处境

### 性格
- 核心性格（3-5个关键词）
- 表面性格 vs 内在性格（他展现给别人的 vs 真实的他）
- 说话方式（用词习惯、句子长短、是否幽默/严肃/反问）
- 习惯性动作

### 三观与利益
- 世界观（他怎么看这个世界）
- 价值观（什么对他最重要）
- 人生观（他想要什么样的生活）
- 核心利益（他在保护什么、追求什么）
- 利益决定立场——他在冲突中会站哪一边？为什么？

### 关系网
- 与主角的关系（及变化方向）
- 与其他重要角色的关系
- 利益冲突点 / 利益共同点

### 人物弧光
- 初始状态（有什么缺陷或错误认知）
- 关键转折点
- 最终成长（或堕落）

### 秘密与弱点
- 不为人知的秘密
- 致命弱点
- 最怕失去的东西
```

### 人物驱动 vs 作者驱动

**人物驱动（正确）**：人物的性格和利益决定了他会做什么。作者的工作是设计好人物，然后让人物自己"活"起来。当两个利益冲突的人物碰到一起，剧情自然就产生了。

**作者驱动（错误）**：作者需要一个转折，就硬安排一个角色做出不符合人设的事情。比如一个聪明谨慎的人突然犯蠢、一个重情重义的人毫无理由地背叛。读者会觉得"假"。

**检验标准：** 写完任何一个人物的行为，问自己——如果我是他，知道他知道的信息，处于他的处境，有他的性格和利益，我会这样做吗？如果答案是"不会"，这个行为就需要修改。

### 对话即人物

每个角色说话的方式应该不同：用词习惯、句子长短、是否爱开玩笑、是否习惯反问。**读者不看名字就应该能分辨出是谁在说话。**

对话不是传声筒——两个人你一句我一句地交代剧情。好的对话有潜台词，说的和想的不一定一样。人物在对话中会回避、会试探、会撒谎、会言不由衷。

### 配角出场原则

不要在第一章堆砌所有人物。配角只在需要他推动情节时才出场。每章出场人物控制在5-7个以内。重要配角可以延迟到第二章、第三章再出场——通过主角的视角逐步认识新人物，保持新鲜感。

## 第三层：剧情设定

世界无比繁杂，其中有一个好故事值得讲给读者听。剧情是从浩瀚的世界观中提取出的一段精彩故事——它应该有跌宕起伏的发展，有宏观世界的展现，有细微生活的体现，和每个人物合乎身份的行为方式，交织在一起。

### 核心原则

**剧情由人物碰撞产生。** 把两个立场不同、利益冲突的人放在同一个场景里，剧情自然就来了。不需要作者强行制造巧合。

**大弧线套小弧线。** 长篇小说需要一条贯穿始终的主线（比如"发现AI社会的真相"），但读者不会因为主线而坚持读下去——他们是被每一章的小冲突、小悬念、小高潮牵着走的。每3章一个小高潮，每10章一个中高潮，每卷一个大高潮。

**因果链条要严密。** 每个事件都有触发原因、发展过程、必然结果、意外转折。事件之间要有因果关系，不是随机堆砌。好的剧情是"A导致B，B引发C，C出乎意料地影响了D"——读者回头看会发现一切都是环环相扣的。

**曲折是必须的。** 禁止直线推进。每一个看似顺利的进展后面都要跟一个意外。希望→挫折→调整→小胜→更大的挫折→绝境→逆转。

### 结构模板

**三幕式（适合中短篇）**：
- 第一幕（25%）：铺垫——展示日常→激励事件→主角踏上旅程
- 第二幕（50%）：对抗——不断尝试→重大转折→最黑暗的时刻
- 第三幕（25%）：解决——最终对决→高潮→新的平衡

**长线叙事（适合长篇连载）**：
每卷有自己的完整弧线（开篇→发展→高潮→收尾），同时为下一卷埋下伏笔。整体主线在每卷中推进一步，但不急于揭示全貌。

### 伏笔设计

伏笔是第三层的核心工具。好的伏笔：
- **隐蔽**：读者第一次看时不察觉
- **合理**：揭开后回头看，发现早有铺垫
- **重要**：对剧情有实质影响

类型包括：物品伏笔（一把剑、一个玉佩）、对话伏笔（"将来你会明白的"）、行为伏笔（某人反常的举动）、环境伏笔（天气、方位变化）。

### 复杂因果关系设计

避免简单因果（A→B），设计复杂因果：

- **多重动机**：每个行为背后有表面原因、深层原因、隐藏原因
- **意外后果**：A想做X，结果导致Y，Y又引发Z
- **利益冲突**：两个合理的选择互相矛盾，人物必须做出艰难抉择
- **认知差异**：同一件事，不同人物有不同理解，导致误会或冲突
- **蝴蝶效应**：小事件引发大后果

## 第四层：讲故事的方法

下面三层都是作者从上帝视角设计出来的。但作者不能在故事中直白地告诉读者这些。**好的故事要娓娓道来，由人物的行为和对话来推动，让读者自己去发现、去品味、去恍然大悟。**

这是四层中最重要的一层——因为它直接决定了读者的阅读体验。前三层再好，讲不好就是浪费。

### 铁律：绝对禁止上帝视角

**读者只能知道叙事视角人物（通常是主角）此刻知道的事情。** 主角不知道的，读者也不能知道。

- 幕后的阴谋、隐藏的势力、真正的敌人——这些必须随着主角的探索一点一点被揭开
- 坚决不要用"他不知道的是……""殊不知……"这种句式——这就是上帝视角剧透
- 不要切换到反派/幕后势力的视角来展示他们的计划（除非这是有意的叙事策略，且不会泄露核心悬念）
- 不要用大段旁白来介绍世界观、交代背景、解释人物动机

**检验标准：** 写完一章后，问自己——如果一个完全不知道故事大纲的读者来看，他能从这一章中感受到悬念和好奇心吗？还是所有秘密都已经被交代了？如果是后者，必须重写。

### 展示而非叙述（Show, Don't Tell）

这是第四层最基础的技法。

**错误**："楚铮很聪明，善于观察。他看出了父亲很担心。"
**正确**："楚铮低着头，目光却悄悄上移。他看见父亲握剑的手指节发白，那是只有紧张时才会有的习惯。"

**错误**："到了2033年，全球92%的车辆使用AI驾驶。"
**正确**：主角叫了一辆车，车自己开过来。他在后座用手机养龙虾。路上经过工地，全是机器人干活。他习以为常。

原则：用动作代替形容词，用细节暗示情绪，用环境烘托气氛，留白让读者推断。

### 绝对禁止画外音（最高优先级）

**画外音 = 作者跳出故事，以旁白身份向读者解释、总结、提示。** 这是最常见也最致命的写作问题。

> **详细规范见 `strict-narration` 技能。** 该技能提供六种画外音检测模式、Deep POV操作规则、MRU因果结构、信息场景化技法、逐句自检流程。**每次写作前必须加载。**

核心自检：每写完一句话问自己——"去掉角色，这句话还能成立吗？"如果能，它就是画外音，重写。

### 智商设定135+

**所有人物和读者的智商设定为135以上。**

- 不写浅显的暗示（"楚铮心中明白，这是功高震主"）
- 不写低劣的总结（"由此可见，楚名棠是个疼爱儿子的父亲"）
- 不写直白的解释（"原来，这一切都是皇帝的阴谋"）
- 删除所有"原来""由此可见""显然""他心中明白""他知道"等词
- 相信读者能自己看懂

### 悬念是故事的生命线

- 每一章必须制造至少一个让读者"想继续看下去"的钩子
- 危险来了但不解释为什么，幸运逃脱但不解释怎么回事——让读者带着疑问翻到下一章
- 伏笔要藏得深：一个看似无关的细节，在十章之后才揭开意义
- "巧合"是好工具——主角觉得是巧合，读者隐约觉得不对，真相要到很后面才揭开
- 让主角一步步发现真相，每一步发现都带来新的疑问

### 主角的专业能力决定发现路径

主角是什么职业，就应该从那个职业的专业视角来发现异常。不要让主角"凑巧"看到一份机密文件。让他用自己的专业能力碰到异常——电力交易员发现AI异常吃电，网络安全工程师发现异常数据流，外卖骑手发现手机信号在某些区域总是中断。

专业发现→好奇心驱动→更深入调查→发现更大的秘密→越陷越深。这是最自然的叙事链条。

### 节奏控制

> **详细场景构建技法见 `scene-craft` 技能。** 该技能提供 Swain 场景结构（目标-冲突-灾变）、对话潜台词四技法、张力递进原则、节奏工具、章末钩子三类型。

**章节内部节奏**：
- 开头（慢）：环境/氛围铺垫
- 发展（快）：事件推进，冲突升级
- 高潮（极快）：冲突爆发，情绪顶点
- 结尾（慢→悬念）：情绪沉淀，留下钩子

**节奏工具**：短句制造紧张，长句营造舒缓。短段落加快节奏，长段落放慢。频繁场景切换加快，单一场景深入放慢。

**时间跳跃**：避免"一个月后，楚铮的武功大进"这种生硬跳跃。用季节变化、人物成长、事件结果、他人反应来自然过渡。

**慢节奏原则**：不要急着推主线。先让读者爱上这座城市、爱上角色的日常生活，再去打破它。一个事件不要在一章内从发生到结束——拆成前兆→酝酿→爆发→反应→余波。

### 场景描写

用五感描写而非形容词堆砌。"房间里很乱"不如"霉味混合着血腥气钻进鼻腔，脚下踩到了什么——低头看去，是一枚断裂的指甲"。

每个场景用具体地名，带一个感官细节（热气、酱汁味、灯光颜色、引擎声），环境描写不超过3句话——点到为止。

## 续写已有小说

续写需要额外完成以下工作：

**深入研读原著**：理解已有的世界观、人物设定、剧情脉络、未回收的伏笔、文风特征。

**确定续写定位**：忠实原著？创新改编？平行世界？从哪个节点接续？

**创作续写大纲**：在原著设定的基础上，设计新增剧情线、人物发展方向、伏笔回收计划。

**保持一致性**：续写最大的挑战是"感觉不像同一个故事"。特别注意人物说话方式、叙事节奏、用词习惯的一致性。

---

## 质量检查：逐层验证

每章写完后，从底层到顶层依次检查：

### 第一层检查（世界观）
- [ ] 涉及的科学/技术/社会常识是否准确？
- [ ] 是否与之前确立的世界设定一致？
- [ ] 有没有违反这个世界的规则？

### 第二层检查（人物）
- [ ] 每个人物的行为是否符合他的人设（性格、立场、利益）？
- [ ] 对话是否符合人物身份（用词、语气、习惯）？
- [ ] 人物情感变化是否有铺垫？
- [ ] 是否存在"工具人"——某个角色纯粹为了推动剧情而行为反常？

### 第三层检查（剧情）
- [ ] 本章有明确的核心事件吗？
- [ ] 事件之间的因果链条是否清晰？
- [ ] 有没有伏笔需要铺设或回收？
- [ ] 剧情节奏是否有起伏？是否在一章内推进过快？

### 第四层检查（讲故事）
- [ ] 有没有上帝视角的信息泄露？
- [ ] 有没有"他不知道的是""殊不知"之类的句式？
- [ ] 是否做到了"展示而非叙述"？
- [ ] 有没有替读者总结、解释、判断？
- [ ] 结尾是否留有悬念或期待点？
- [ ] 对话占比是否合理？有没有长段落的心理描写或背景介绍？
- [ ] 场景描写是否用了感官细节？

### 跨层一致性
- [ ] 第四层的呈现有没有暴露读者不该知道的第一/二/三层信息？
- [ ] 人物的行为（第二层）有没有与世界规则（第一层）矛盾？
- [ ] 剧情（第三层）的推进有没有让某个人物做出违反人设（第二层）的事？

## 爽点节奏（异能/金手指类小说必读）

**核心原则：异能必须让读者羡慕。** 如果你的角色有超能力，读者在每一章都应该至少有一个瞬间想"我也想要这个能力"。异能不能只用于主线剧情——它必须渗透到日常生活中，让普通场景变得不普通。

**爽点的三种类型：**

**1. 日常便利爽。** 用异能解决生活中的问题，做到普通人做不到的事。不需要多大的场面——关键是"普通人做不到，但我能做到"的对比。

示例：
- 外卖骑手用地脉感知避开堵车路段（地下有裂缝=地面在施工），比导航还快
- 感知生命的人在小贩中心一眼看出哪个摊位的食材最新鲜（食材的"生命力"）
- 护身石头在主人被球砸到脸之前弹出一层薄光，球弹开了，旁边的人一脸懵

**2. 路见不平爽。** 用异能帮助别人、化解危险。旁人不知道发生了什么，但读者知道——这种"暗中拯救"的感觉极其上瘾。

示例：
- 感知到一个老人心跳异常，在他倒下之前扶住他，叫了救护车
- 掌心感知到脚下的水管即将爆裂，把路过的小孩拉开，三秒后水柱喷出
- 护身石头在地震中包住了一个孩子，碎玻璃砸在金色的光罩上滑下去

**3. 打脸反转爽。** 有人质疑、轻视、挑衅——然后主角用能力做了一件不可能的事——质疑者闭嘴。不需要嘴炮。行动就是最好的打脸。

示例：
- "你一个送外卖的懂什么？"——林启把手按在地上，告诉你地下三十米的水管布局、裂缝位置、哪栋楼的地基有问题——比工程师的仪器还准
- "你那个直觉不靠谱"——里扎尔闭眼指了三个方向：那边有个孕妇、那边有个骨折的人、那边有个心律不齐的——全对

**爽点节奏规则：**
- 每2-3章必须有一个异能爽点场景
- 爽点分大小：小爽点=日常便利（每2章一次），大爽点=危机化解/打脸（每5章一次）
- 前10章至少需要3-4个小爽点+1个大爽点
- 爽点不能只出现在主角身上——每个觉醒者都要有自己的高光时刻
- 爽点场景必须有旁观者（哪怕只有一个人）——没人看到的神操作不是爽点
- 用细节制造"普通人做不到"的对比：旁人的反应、事后的验证、其他方式的不可能性

**爽点与"不画外音"的平衡：**
- 不需要旁白说"林启用息壤封住了裂缝，众人惊叹"
- 直接写：林启的手按在地上——金色的光从指缝流出来——裂缝的边缘在动——在合拢——旁边的工人放下了锹——看着地面——裂缝没了——他蹲下来摸了一下——平的——"刚才……这里不是裂了吗？"
- 旁观者的困惑和震撼通过他们的动作和台词传达，不需要作者解释

---

## 卡文时怎么办

1. **回顾第三层大纲**：是否偏离了原定路线？
2. **回到第二层**：想想每个在场人物此刻的利益和情绪，让人物碰撞产生新的可能
3. **制造冲突**：引入新角色或新事件
4. **跳着写**：先写后面的高潮情节，再回来填中间
5. **休息**：暂时放下

---

## 总结

四层框架的本质是一句话：**自底向上构建，自顶向下呈现。**

作者的工作是在读者看不见的地方搭好地基（世界观）、立好柱子（人物）、架好梁（剧情），然后用精妙的讲述方法盖上屋顶——读者走进来，看到的是一个浑然天成的空间，而不是裸露的钢筋水泥。

永远记住：读者不需要看到你的设计图纸。他们需要的是住在故事里、不想出来的体验。
