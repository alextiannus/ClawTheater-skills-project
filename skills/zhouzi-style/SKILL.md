# 肘子风格小说创作指南

## 铁律（违反任何一条即为失败）

1. **禁止内心独白**：不写"他心想"、"他觉得"、"他暗自思索"。一切通过行为、对话、系统交互表达
2. **禁止解释性叙述**：不写"他之所以这样做是因为"、"原来如此"。读者智商135+
3. **每章必须有笑点**：至少3个让人发笑的对话或场景
4. **每章必须有钩子**：结尾必须留悬念，让读者想看下一章

## 核心文风特征

### 主角人设公式

```
表面：人畜无害 + 一本正经
内在：毒舌吐槽王 + 气人于无形
行为：把所有严肃的事情往荒诞方向带
```

主角特征清单：
- 遇到生死危机先问"有钱拿吗？"
- 对强者的威胁回应以"谢谢夸奖"
- 永远把神级能力用在最接地气的事上（控水送外卖、龙息热牛奶）
- 在所有人都紧张的时候，问一个完全不相关的问题

### 系统交互模板

系统是主角的"吐槽搭档"（漫才中的ツッコミ役），不是冷冰冰的面板：

```markdown
# 标准互动节奏

主角说蠢话 → 系统认真回应 → 主角把话题带偏 → 系统沉默/无语

# 示例
"系统，我能用龙息烤串吗？"
【应龙血脉是上古战斗之力，不是烧烤工具。】
"但我饿了。"
【……】
```

系统的人格特点：
- 用【】括号标注，简洁冷静
- 被主角"整无语"后会沉默（用【……】表达）
- 偶尔吐槽主角（"宿主心态真好"）
- 永远认真回答荒谬的问题

### "谢谢夸奖"公式

这是肘子风格最标志性的对话模式：

```
角色A：你很[某种看似负面的评价]。
主角：谢谢夸奖。
角色A：这不是夸奖！

变体1：
角色A：你真的很奇怪。
主角：谢谢夸奖。
角色A：这不是夸奖。

变体2：
角色A：你是个天才。
主角：谢谢夸奖。
角色A：这次是真的夸奖。（反转）
```

**使用频率**：每2-3章出现一次，不要每章都用，避免审美疲劳。每次出现时略微变化措辞或情境。

### 对话节奏

肘子风格的对话特点是"快、短、密"：

```markdown
# 正确（肘子风格）
"为什么？"
"因为你很讨厌。"
"……"
林昭摸了摸鼻子："那我们今天练什么？"

# 错误（传统网文风格）
林昭疑惑地问道："为什么不想听我说话呢？我觉得我们可以好好沟通一下。"
苏晚棠冷冷地看着他，过了好一会儿才开口："因为你这个人实在是太让人讨厌了，我一看到你就觉得很烦躁。"
```

**节奏原则**：
- 对话占全文60-70%
- 单句对话控制在20字以内
- 动作描写穿插在对话间，一句话搞定
- 少用"说道"、"回答道"，多用动作替代（"他挑了挑眉"、"她冷笑"）

## 情节构建

### 章节结构（3000-5000字/章）

```
开场（承接+引子）    → 300-500字
发展（对话+推进）    → 1500-2500字（核心段落，笑点密集）
高潮（冲突/反转）    → 800-1200字
钩子（悬念结尾）     → 200-300字
```

### 笑点设计模板

| 类型 | 说明 | 使用频率 |
|------|------|----------|
| 错位反差 | 神级能力做low事（龙息烤串） | 每章1-2次 |
| 系统无语 | 主角把系统整沉默 | 每章1次 |
| 谢谢夸奖 | 经典互动公式 | 每2-3章1次 |
| 一本正经胡说八道 | 认真语气+荒谬内容 | 每章1-2次 |
| 气人到破防 | 把配角气到失态 | 每2章1次 |
| 关键时刻跑题 | 生死关头问不相关的事 | 每3章1次 |

### 战斗场景原则

- 战斗中穿插对话和吐槽，**禁止纯打斗超过500字**
- 主角打赢后说"谢谢夸奖"或问一个荒谬的问题
- "扮猪吃老虎"是核心爽点：对手小看主角 → 主角用奇招反杀
- 胜利靠智慧和环境，不靠硬实力碾压

### 感情线原则

- **禁止甜腻**：男女主的互动必须建立在"互怼"基础上
- 女主被气到破防 = 好感度+1
- 主角永远不主动表白，通过行为展现关心
- 多女角色时，她们共同的特点是"被主角气到"

## 叙事节奏与场景密度（重要）

### 慢节奏原则

剧情发展速度必须放慢至少5倍。不要急着推主线——先让读者爱上这座城市、爱上角色的日常，再去毁灭它。

**核心思路**：一个事件（如"天空裂开"）不要在一章内从发生到结束。把它拆成：前兆→酝酿→爆发→反应→余波，每个阶段至少一章。

**章节内容分配**：
- 主线剧情推进：不超过全章的30%
- 日常场景、人物互动、环境描写：至少占50%
- 伏笔铺设和氛围营造：约20%

**禁止事项**：
- 禁止在一章内完成"发现问题→解决问题"的完整闭环
- 禁止角色一见面就交代完所有背景信息
- 禁止跳过"从A地到B地"的过程——路上才是写故事的好时机

### 场景描写要求

每章至少包含2-3个具体的场景描写段落，用感官细节而非形容词堆砌：

```markdown
# 正确
牛车水的史密斯街上，两排骑楼之间夹着一条窄得只能并排走两个人的巷子。
地面上的热气能把拖鞋底烤软。

# 错误
牛车水是一个非常繁华热闹的华人聚居区。
```

**场景细节原则**：
- 用具体的地名（史密斯街、克拉码头、大巴窑第23座），不用泛泛的"市中心"
- 每个场景带一个感官细节：热气、酱汁味、灯光颜色、引擎声
- 环境描写不超过3句话——点到为止，不写游记

### 新加坡市井日常

故事发生在新加坡，必须让读者闻到这座城市的味道：

- **巴刹（菜市场）**：凌晨五点开档的鱼贩、挑菜的阿姨、湿漉漉的地面
- **组屋（HDB）**：楼下架空层下棋的大叔、电梯里的"不许吃榴莲"告示、走廊尽头晾的校服
- **小贩中心**：海南鸡饭、肉骨茶、拉茶；档口老板和食客的对话是最好的世界观容器
- **交通**：地铁的报站声、出租车司机的口音、外卖骑手在车流中穿梭
- **天气**：赤道的暴晒、突如其来的雷阵雨、空调室内外的温差

这些日常不是"填充物"——它们是读者爱上这个故事的理由。灾难来临时，读者会因为记得鸡饭陈的档口而心痛。

### 路人视角穿插

每2-3章穿插一个路人视角的短场景（300-800字），用普通市民的眼睛侧写异变：

- 出租车司机发现GPS开始跳字
- 7-Eleven的夜班店员看到冰柜里的水全部结冰了
- HDB的保安在监控画面里看到楼道里有"什么东西"一闪而过
- 一个大妈发现今天买的鱼——眼睛在发光

**路人视角的作用**：
- 增加真实感和恐惧感——普通人的视角比超能力者更能传递危机的重量
- 暗示世界观的扩散——异变不只发生在主角身边
- 制造悬念——路人看到的片段读者无法完全理解，激发好奇心
- 让城市成为一个"角色"——有血有肉的、正在被改变的城市

### 反派内部视角

每3-5章穿插一段天枢局内部的短场景（500-1000字）：

- 监控室分析员发现异常数据
- 执行官之间对"格式化"政策的私下讨论
- 指挥官"天枢"的个人故事片段
- 阿莎在叛逃之前的道德困境时刻

**反派视角的原则**：
- 天枢局不是纯恶组织——他们有自己的逻辑和恐惧
- 执行官也是人——有家庭、有犹豫、有在深夜失眠的时候
- 用"他们也觉得自己是对的"来制造真正的戏剧张力
- 禁止脸谱化反派——没有一个角色该是"为了作恶而作恶"的

### 角色前史与日常

每个核心角色在觉醒前，至少花一个完整场景展示他们的日常生活：

- **林启**：三份工的轮转、SP的课堂、凉掉的外卖就是他的晚饭
- **老周**：996的加班、和产品经理的拉扯、深夜独自debug的孤独感
- **里扎尔**：外卖路线就是他的社交网络、他认识这座城市的每一条近路
- **阿凯**：补习班、独自等人的公园长椅、把一份鸡饭省出半份当明天午饭

这些前史不是"闪回"——它们就是故事本身。读者先认识"人"，再接受"超能力"。

### 故事曲折度

剧情发展必须更加曲折。禁止直线推进——每一个看似顺利的进展后面都要跟一个意外：

- 刚找到一个觉醒者——结果他拒绝加入
- 刚制定好计划——情报过时了，天枢局提前行动
- 刚打赢一场——发现胜利带来了新的问题
- 刚建立信任——有人的动机受到质疑

**节奏公式**：希望→挫折→调整→小胜→更大的挫折→绝境→逆转

## 质量检查清单

每章写完后自查：

- [ ] 是否有内心独白？（删掉）
- [ ] 对话是否占50%以上？（慢节奏下可降至50%，留空间给场景描写）
- [ ] 是否有至少2个笑点？（慢节奏下可减至2个，避免密度过高）
- [ ] 结尾是否有钩子/悬念？
- [ ] 主角是否"一本正经地气人"了？
- [ ] 是否有解释性叙述？（删掉）
- [ ] 是否有至少2段具体的场景/环境描写？
- [ ] 是否有路人或反派的视角穿插？（每2-3章至少一次）
- [ ] 角色的日常生活细节是否足够？
- [ ] 剧情是否在一章内推进过快？（如果是，拆成两章）
- [ ] 配角是否有自己的性格标签？
